## Autonomous Aerospace Intelligence Web Agent Powered by TinyFish

---

## 🌍 Overview

OrbitScout AI is an autonomous aerospace intelligence platform that continuously monitors the global aerospace ecosystem and transforms scattered web information into structured, actionable insights.

The platform leverages the TinyFish Web Agent API to navigate real aerospace websites, government procurement portals, and space agency platforms to detect:

- Satellite launches
- Aerospace tenders
- Defense contracts
- Aviation regulations
- Space mission announcements

Instead of manually monitoring hundreds of websites, OrbitScout deploys an AI-powered web agent that autonomously performs multi-step workflows on the live web and generates intelligence reports.

This project demonstrates how agentic infrastructure can convert the web into an executable runtime for AI agents, enabling real-world autonomous work.

---

## Problem Statement

The aerospace and defense industry is highly information-driven. Critical intelligence signals are distributed across hundreds of different sources including:

- Space agency portals
- Government procurement platforms
- Aerospace company announcements
- Aviation regulatory agencies
- Research institutions

Examples include:
- satellite launch announcements
- aerospace procurement opportunities
- defense contracts

Currently, companies rely on manual monitoring and fragmented intelligence gathering.

This leads to:

There is no autonomous system capable of continuously navigating the live web and collecting aerospace intelligence in real time.

## 📦 Installation

### Clone Repository

```
git clone https://github.com/YOUR_USERNAME/orbitscout-ai
```

---

### Backend Setup

```
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

Backend runs on:

```
http://localhost:8000
```

---

### Frontend Setup

```
cd frontend
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:3000
```
